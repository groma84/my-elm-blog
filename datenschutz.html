Log-Dateien

Bei jedem Aufruf wird eine anonymisierte IP-Adresse gespeichert - der letzte Block fehlt jeweils, so dass ein Rückschluss auf einen Computer bzw. ein Gerät nicht möglich ist. Außerdem wird gespeichert, mit welchem Browser welche Adresse aufgerufen wurde. Alle Daten werden 14 Tage gespeichert und dann automatisch gelöscht. Eine Auswertung der Daten findet nicht statt.
